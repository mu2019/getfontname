getfontname

get ttf/ttc font file font name

View on GitHub Download .zip Download .tar.gz

ReadMe

使用纯python无需第三方库即可实现读取ttc/ttf字体文件的字体名称. get ttf/ttc font file font name

How

from getfontname import OTFName

ttf=OTFName('c:/windows/fonts/msyh.ttf',langid=zh_cn)
print('font file :',ttf.FontFile) print('font en family',ttf.getFontInfo('Family')) print('font local family',ttf.getFontLocalInfo('Family'))

out put

font file : c:/windows/fonts/msyh.ttf

font en family ['Microsoft YaHei']

font local family ['微软雅黑']
